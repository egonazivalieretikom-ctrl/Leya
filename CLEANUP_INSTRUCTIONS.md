# Инструкция по очистке репозитория Леи

Этот документ описывает процесс санитарной очистки репозитория от дубликатов, мусора и секретов.

## ⚠️ КРИТИЧЕСКИ ВАЖНО

Текущий репозиторий содержит:
- **Дубликаты исходников** в `leya_brain/` и `leya_soul/` (полные копии LeyaOS.py, .env, JSON)
- **UUID-папки** (временные сессии, не очищенные)
- **`.env` в Git** (секреты и API ключи)
- **Pickle файлы** в Git (бинарники, риск безопасности)

Это создает риск модификации собственного кода Леи в рантайме и утечки секретов.
